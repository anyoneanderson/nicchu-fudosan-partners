# 日中不動産パートナーズ ホームページ修正ガイド

---

## Part 1: 初期セットアップ（セットアップ担当者が実施）

### Step 1: 必要なアカウント作成

#### GitHub アカウント
1. https://github.com にアクセス
2. 「Sign up」→ メールアドレス・パスワード・ユーザー名を入力
3. メール認証を完了する
4. 作成したアカウント名を控えておく → **リポジトリへの招待に使う**

#### Google アカウント
- 既にGmailを持っていればそれでOK
- なければ https://accounts.google.com で作成

---

### Step 2: リポジトリへの招待

1. https://github.com/anyoneanderson/nicchu-fudosan-partners/settings/access にアクセス
2. 「Add people」→ 作成したGitHubアカウント名を入力
3. 権限は「Write」を選択
4. 招待を送信 → **相手側でメールから承認してもらう**

---

### Step 3: Node.js のインストール

1. https://nodejs.org にアクセス
2. **LTS（推奨版）** をダウンロード
3. インストーラーを実行（すべてデフォルト設定でOK）
4. 確認：ターミナル（Macならターミナル.app）を開いて以下を実行

```
node -v
```

バージョン番号（例: `v22.xx.x`）が表示されればOK。

---

### Step 4: Google Antigravity のインストール

1. https://antigravity.dev にアクセス
2. 使用中のOS用インストーラーをダウンロード
3. インストーラーを実行

#### 日本語化の設定
1. Antigravity を起動
2. 左サイドバーの拡張機能アイコン（四角が4つ）をクリック
3. 検索欄に「Japanese」と入力
4. 「Japanese Language Pack for VS Code」をインストール
5. 右下に表示される「Restart」をクリック → メニューが日本語になる

---

### Step 5: Git の初期設定

Antigravity のターミナル（画面下部）で以下を実行：

```
git config --global user.name "名前をここに入力"
git config --global user.email "GitHubに登録したメールアドレス"
```

---

### Step 6: プロジェクトのクローンと準備

Antigravity のターミナルで以下を順番に実行：

```
cd ~/Desktop
git clone https://github.com/anyoneanderson/nicchu-fudosan-partners.git
cd nicchu-fudosan-partners
npm install
```

デスクトップに「nicchu-fudosan-partners」フォルダができます。

---

### Step 7: 動作確認

ターミナルで以下を実行：

```
npm run dev
```

表示される `http://localhost:3000` をブラウザで開いてサイトが表示されればセットアップ完了！

止めるときはターミナルで `Ctrl + C` を押す。

---

## Part 2: 日常の修正作業（作業者向け）

### ホームページを修正する流れ

```
1. Antigravity を開く
2. プレビューを起動する
3. AIに修正を頼む
4. プレビューで確認する
5. 変更を保存・アップロードする
```

---

### 1. Antigravity を開く

1. Antigravity を起動
2. 「ファイル」→「フォルダーを開く」
3. デスクトップの「nicchu-fudosan-partners」フォルダを選択

---

### 2. プレビューを起動する

画面下部のターミナルに以下を入力して Enter：

```
npm run dev
```

ブラウザで http://localhost:3000 を開くとサイトが見える。
これを開いたまま作業すると、修正が即座にプレビューに反映される。

---

### 3. AIに修正を頼む

画面右側のAIチャット欄に、**日本語でやりたいことを書く**だけ！

#### 書き方の例

| やりたいこと | AIへの指示の例 |
|------------|--------------|
| 電話番号を変えたい | 「トップページの電話番号を 03-XXXX-XXXX に変更して」 |
| 会社の住所を変えたい | 「会社概要の住所を○○に変更して」 |
| 写真を差し替えたい | 「トップページのメイン画像をこのファイルに差し替えて」 |
| 文章を修正したい | 「サービス紹介の説明文を以下に変更して：○○○○」 |
| 新しいページを追加 | 「採用情報ページを新しく作って。内容は○○」 |

**コツ：**
- 「どのページの」「何を」「どう変えたいか」を具体的に書く
- 一度にたくさん頼むより、1つずつ頼む方が確実
- 修正後にプレビュー（ブラウザ）で確認してからOKを出す

---

### 4. プレビューで確認する

- AIが修正すると、ブラウザのプレビューが自動で更新される
- 日本語ページ: http://localhost:3000/ja/
- 中国語ページ: http://localhost:3000/zh/
- 問題なければ次のステップへ
- やり直したい場合は「元に戻して」とAIに言えばOK

---

### 5. 変更を保存・アップロードする

修正が完了したら、変更をGitHubにアップロードする。

#### 方法A: AIに頼む（簡単）

チャット欄に以下のように入力：

```
変更をコミットしてプッシュして。メッセージは「電話番号を変更」で。
```

#### 方法B: 画面の操作で行う

1. 左サイドバーの分岐アイコン（ソース管理）をクリック
2. 変更されたファイルが一覧表示される
3. 上部のメッセージ欄に変更内容を入力（例：「電話番号を変更」）
4. 「✓ コミット」ボタンをクリック
5. 「変更の同期」ボタンをクリック

→ しばらくするとNetlifyが自動でサイトを更新してくれる。

---

### 作業を終えるとき

1. ターミナルで `Ctrl + C` を押してプレビューを停止
2. Antigravity を閉じる

---

## 困ったときは

| 症状 | 対処法 |
|------|--------|
| プレビューが表示されない | ターミナルで `npm run dev` を再実行 |
| AIが英語で返答する | 「日本語で回答してください」と追加で伝える |
| 変更が反映されない | ブラウザでページを更新（F5 または Cmd+R） |
| よくわからないエラーが出た | スクリーンショットを撮ってセットアップ担当者に送る |
| 元に戻したい | AIに「直前の変更を元に戻して」と頼む |
