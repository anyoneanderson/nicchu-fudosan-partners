# 要件定義書 - 日中不動産パートナーズ株式会社 コーポレートサイト

## 1. 概要

### 1.1 プロジェクトの目的

日中不動産パートナーズ株式会社のコーポレートサイトを構築する。中国人投資家向けの日本不動産投資コンサルティング・仲介事業を展開する企業として、日本語・中国語の2言語で事業内容と信頼性を訴求し、問い合わせ獲得を目指す。

### 1.2 背景

- 中国人投資家の日本不動産への関心が高まっている
- 日中両言語で事業内容を発信し、投資家との接点を増やす必要がある
- 最新のWebデザイントレンドとアニメーションを活用し、先進的な企業イメージを確立する

### 1.3 ターゲットユーザー

| ユーザー | 言語 | 主な利用目的 |
|---------|------|-------------|
| 中国人投資家 | 中国語 | 日本不動産への投資検討、サービス内容の確認、問い合わせ |
| 日本国内の取引先・パートナー | 日本語 | 会社情報の確認、提携の検討 |
| 日本在住の中国人 | 日本語/中国語 | 不動産購入・賃貸の相談 |

## 2. 機能要件

### ページ構成

[REQ-001] トップページ
- ファーストビューにインパクトのあるアニメーション演出
- 事業コンセプトのキャッチコピー（日本語/中国語）
- サービスの概要紹介セクション（投資コンサル・仲介・内見サポート）
- 問い合わせへの導線（CTA ボタン）
- 言語切り替えナビゲーション（/ja/ ⇄ /zh/）

[REQ-002] 会社概要ページ
- 会社名（日中不動産パートナーズ株式会社）、所在地、設立日、代表者名（太田翔子）などの基本情報
- 事業内容の説明
- 企業理念・ミッション
- 沿革（必要に応じて）

[REQ-003] お問い合わせページ
- Netlify Forms を利用した問い合わせフォーム
- 入力項目: 氏名、メールアドレス、電話番号（任意）、問い合わせ種別、問い合わせ内容
- バリデーション（必須項目チェック、メール形式チェック）
- 送信完了メッセージの表示
- スパム対策（Netlify のハニーポット機能）

### 多言語対応

[REQ-004] 日本語・中国語の2言語対応
- `/ja/` パスで日本語ページを表示
- `/zh/` パスで中国語ページを表示
- ルートパス（`/`）へのアクセスは `/ja/` にリダイレクト
- ヘッダーに言語切り替えリンクを常時表示
- 各言語で同一のページ構成を維持

### 共通UI

[REQ-005] ヘッダー
- 会社ロゴ
- ナビゲーションメニュー（トップ、会社概要、お問い合わせ）
- 言語切り替えボタン（日本語/中文）

[REQ-006] フッター
- 会社基本情報（社名、住所、連絡先）
- コピーライト表記
- 各ページへのリンク

### アニメーション・デザイン

[REQ-007] モダンアニメーション演出
- Remotion または同等のアニメーションライブラリを活用
- スクロールトリガーによるセクション表示アニメーション
- ファーストビューのダイナミックな演出（パーティクル、パララックス等）
- ページ遷移時のスムーズなトランジション
- パフォーマンスに配慮した実装（重いアニメーションの遅延読み込み等）

### 画像素材生成

[REQ-008] Gemini API による画像素材の自動生成
- google-genai SDK を使った Python スクリプトで画像を生成
- ヒーロー背景、サービスイラスト、OGP画像を一括生成
- プロンプト定義ファイル（prompts.json）で画像内容を管理
- API キー未設定時はプレースホルダー（CSS グラデーション）で代替
- 後から手動で画像を差し替え可能

## 3. 非機能要件

[NFR-001] パフォーマンス
- Lighthouse スコア 90+ を目標（Performance, Accessibility, Best Practices, SEO）
- 静的サイト生成（SSG）により高速な初期表示を実現
- アニメーションは GPU アクセラレーションを活用し、60fps を維持

[NFR-002] レスポンシブデザイン
- モバイル（375px〜）、タブレット（768px〜）、デスクトップ（1024px〜）に対応
- モバイルファーストのデザインアプローチ

[NFR-003] SEO
- 各言語ページに適切な `<html lang>` 属性を設定
- `hreflang` タグによる言語関係の明示
- 適切な OGP メタタグの設定
- サイトマップ自動生成

[NFR-004] アクセシビリティ
- セマンティックな HTML 構造
- アニメーションの `prefers-reduced-motion` 対応
- 十分なカラーコントラスト比の確保

## 4. 制約事項

[CON-001] ホスティング
- Netlify 無料枠を使用
- サーバーサイド処理は使用しない（SSG のみ）
- フォーム送信は Netlify Forms の月100件制限に準拠

[CON-002] 技術スタック
- フレームワーク: Next.js（Static Export）
- スタイリング: 未定（Tailwind CSS 推奨）
- アニメーション: Remotion または Framer Motion 等
- フォーム: Netlify Forms
- データベース: 使用しない

[CON-003] コスト
- ホスティング費用ゼロを維持（Netlify 無料枠内）
- 有料サービス・API は使用しない

## 5. 前提条件

[ASM-001] コンテンツ
- 会社名: 日中不動産パートナーズ株式会社
- 代表者名: 太田翔子
- 所在地、設立日等の詳細は別途確認
- 日本語・中国語のテキストコンテンツはクライアントが用意する
- ロゴ・画像素材はクライアントが用意する（仮のプレースホルダーで初期構築）

[ASM-002] ドメイン
- 独自ドメインの取得・設定は別途対応（初期は Netlify のサブドメインで公開可能）

[ASM-003] 運用
- コンテンツ更新は開発者がコードを修正してデプロイする（CMS は使用しない）

## 6. 用語集

| 用語 | 定義 |
|------|------|
| SSG (Static Site Generation) | ビルド時にHTMLを生成する静的サイト生成方式 |
| Netlify Forms | Netlify が提供するサーバーレスフォーム処理機能 |
| Remotion | React ベースの動画・アニメーション生成ライブラリ |
| Framer Motion | React 向けアニメーションライブラリ |
| hreflang | 検索エンジンに多言語ページの対応関係を伝える HTML 属性 |
| OGP (Open Graph Protocol) | SNS でシェアされた際の表示情報を制御するメタデータ |
